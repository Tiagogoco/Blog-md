# Ideas rapidas para tags

Las etiquetas ayudan cuando son pocas, claras y consistentes.

Si cada post tiene tags distintos o casi repetidos, dejan de servir como herramienta de navegacion.

Una buena practica es mantener una lista corta de categorias frecuentes, por ejemplo:

- arquitectura
- frontend
- markdown
- productividad
- notas

Luego, solo si hace falta, agregar etiquetas mas especificas.

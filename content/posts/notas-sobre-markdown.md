# Notas sobre Markdown

Markdown sigue siendo una de las formas mas directas de escribir contenido sin distraerse con herramientas pesadas.

Permite enfocarse en la estructura:

- Titulos
- Parrafos
- Listas
- Enlaces
- Bloques de codigo

Para un proyecto de blog pequeno o mediano, esta simplicidad es una ventaja tecnica. El contenido se puede versionar, mover, revisar y reutilizar con facilidad.

Tambien es una buena opcion cuando el equipo quiere mantener el control de los archivos dentro del mismo repositorio.
